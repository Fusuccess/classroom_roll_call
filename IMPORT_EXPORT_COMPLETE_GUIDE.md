## 快速开始

### 导出数据

1. 打开应用 → 班级管理页面
2. 点击班级的菜单按钮
3. 选择"导出数据"
4. 选择导出类型：
   - 导出学生名单
   - 导出点名记录
   - 生成统计报告
5. 文件自动保存到 Download 文件夹

### 导入数据

1. 打开应用 → 班级管理页面
2. 点击底部的"导入"按钮
3. 输入班级名称
4. 点击"选择 CSV 文件"
5. 系统自动匹配"学生姓名"和"学号"列
6. 预览数据
7. 点击"确认导入"完成导入

## 导出功能

### 导出学生名单

**功能说明：**
- 导出班级中所有学生的信息
- 包含：班级名称、学生姓名、学号、被点名次数、平均分

**文件格式：**
```csv
班级名称,学生姓名,学号,被点名次数,平均分
高一(1)班,张三,001,5,4.20
高一(1)班,李四,002,3,3.67
高一(1)班,王五,003,0,0.00
```

**文件名格式：**
```
students_班级名_时间戳.csv
```

**示例：**
```
students_高一(1)班_1773133345150.csv
```

### 导出点名记录

**功能说明：**
- 导出班级的所有点名记录
- 包含：班级名称、学生姓名、学号、点名时间、评分、备注

**文件格式：**
```csv
班级名称,学生姓名,学号,点名时间,评分,备注
高一(1)班,张三,001,2026-03-10 10:30:45,5,
高一(1)班,李四,002,2026-03-10 10:31:20,未评分,
高一(1)班,王五,003,2026-03-10 10:32:00,4,
```

**文件名格式：**
```
records_班级名_时间戳.csv
```

**示例：**
```
records_高一(1)班_1773133345150.csv
```

### 生成统计报告

**功能说明：**
- 生成班级的统计分析报告
- 包含：班级统计、学生排名、总体数据

**文件格式：**
```csv
班级统计报告

班级名称,高一(1)班
生成时间,2026-03-10 14:30:00

总体统计
总学生数,50
参与点名学生数,45
总点名次数,120
已评分次数,115
平均分,4.15

学生排名
排名,学生姓名,学号,被点名次数,平均分
1,张三,001,5,4.80
2,李四,002,4,4.50
3,王五,003,3,4.33
```

**文件名格式：**
```
report_班级名_时间戳.csv
```

**示例：**
```
report_高一(1)班_1773133345150.csv
```

## 导入功能

### 导入要求

**必须包含的列：**
- **学生姓名** - 包含学生的姓名
- **学号** - 包含学生的学号

**支持的列名：**
- 中文：学生姓名、姓名、名字、学号、号、学生号
- 英文：name、student name、id、student id

### 支持的 CSV 格式

#### 标准格式
```csv
学生姓名,学号
张三,001
李四,002
王五,003
```

#### 多列格式（来自其他系统）
```csv
序号,学生姓名,班级,学号,性别,出生日期
1,张三,高一(1)班,001,男,2008-01-15
2,李四,高一(1)班,002,女,2008-02-20
3,王五,高一(1)班,003,男,2008-03-10
```

#### 不同列顺序
```csv
学号,姓名,班级
001,张三,高一(1)班
002,李四,高一(1)班
003,王五,高一(1)班
```

#### 英文列名
```csv
Student Name,Student ID
Zhang San,001
Li Si,002
Wang Wu,003
```

### 导入步骤

1. **打开导入对话框**
   - 进入班级管理页面
   - 点击底部的"导入"按钮

2. **输入班级名称**
   - 在文本框中输入班级名称
   - 例如：高一(1)班

3. **选择 CSV 文件**
   - 点击"选择 CSV 文件"按钮
   - 从文件管理器中选择文件

4. **自动匹配列**
   - 系统自动识别"学生姓名"和"学号"列
   - 无需手动选择

5. **预览数据**
   - 系统显示导入的学生列表
   - 检查数据是否正确

6. **确认导入**
   - 点击"确认导入"按钮
   - 完成导入

## 常见问题

### 导出相关

**Q: 导出的文件在哪里？**

A: 文件保存在手机的 Download（下载）文件夹中。
- Android：打开文件管理器，进入 Download 文件夹
- iOS：在 Files 应用中查看

**Q: 导出文件会占用多少空间？**

A: 取决于数据量。通常：
- 100 个学生：约 5KB
- 1000 个学生：约 50KB
- 10000 个学生：约 500KB

**Q: 可以导出多少个学生？**

A: 理论上没有限制，但建议：
- 单个班级不超过 200 个学生
- 单个导出文件不超过 10MB

**Q: 导出的数据包含隐私信息吗？**

A: 导出的是学生学号和姓名，不包含其他隐私信息。

### 导入相关

**Q: CSV 文件必须有哪些列？**

A: 必须包含"学生姓名"和"学号"列。其他列会被忽略。

**Q: 列名必须是中文吗？**

A: 不一定。系统支持中文和英文列名：
- 中文：学生姓名、姓名、名字、学号、号
- 英文：name、student name、id、student id

**Q: 如果没有标题行怎么办？**

A: 系统会使用第一行作为标题。如果第一行是数据，请在 Excel 中添加标题行后再导入。

**Q: 如果学生姓名和学号在同一列怎么办？**

A: 不支持。学生姓名和学号必须在不同的列中。

**Q: 导入失败怎么办？**

A: 检查以下几点：
1. 确保有"学生姓名"列和"学号"列
2. 确保学生姓名和学号不为空
3. 确保文件格式正确
4. 查看错误提示信息

**Q: 可以导入多少个学生？**

A: 理论上没有限制，但建议：
- 单个班级不超过 200 个学生
- 单个 CSV 文件不超过 10MB

**Q: 导入后可以修改学生信息吗？**

A: 可以。导入后，你可以在班级详情页面编辑学生信息。

**Q: 导入会覆盖现有数据吗？**

A: 不会。导入是添加操作，会创建新的班级和学生记录。

**Q: 可以导入 Excel 文件吗？**

A: 目前只支持 CSV 格式。但 Excel 可以导出为 CSV：
1. 在 Excel 中打开文件
2. 点击"另存为"
3. 选择"CSV (逗号分隔)"格式
4. 保存文件
5. 导入 CSV 文件

### 编辑和分享

**Q: 可以编辑导出的文件吗？**

A: 可以。导出的是标准 CSV 文件，可以用任何文本编辑器或 Excel 编辑。

**Q: 编辑后的文件可以导入吗？**

A: 可以。编辑后的 CSV 文件可以通过导入功能导入到应用中。

**Q: 可以分享导出的文件吗？**

A: 可以。导出的文件可以通过邮件、云存储等方式分享。

## 技术细节

### 导出服务

**导出学生名单：**
```dart
final filePath = await ImportExportService.exportStudentsToCSV(
  students,
  className,
);
```

**导出点名记录：**
```dart
final filePath = await ImportExportService.exportRecordsToCSV(
  records,
  className,
);
```

**生成统计报告：**
```dart
final filePath = await ImportExportService.generateStatisticsReport(
  students,
  records,
  className,
);
```

### 导入服务

**选择 CSV 文件：**
```dart
final filePath = await ImportExportService.pickCSVFile();
```

**自动匹配列：**
```dart
final (nameColumnIndex, idColumnIndex) = 
  await ImportExportService.autoMatchColumns(filePath);
```

**导入班级和学生：**
```dart
final (classGroup, students) = 
  await ImportExportService.importClassAndStudentsFromCSV(
    filePath,
    className,
    classId,
    nameColumnIndex: nameColumnIndex,
    studentIdColumnIndex: idColumnIndex,
  );
```

### 列名匹配规则

**学生姓名列：**
- 中文：学生姓名、姓名、名字、学生名字
- 英文：name、student name、student_name、full name

**学号列：**
- 中文：学号、号、学生号、学号码
- 英文：id、student id、student_id、number

### 数据验证

- 检查列索引是否有效
- 检查学生姓名是否为空
- 检查学号是否为空
- 自动去除空格
- 自动跳过空行

## 故障排除

### 导出问题

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 找不到导出文件 | 文件保存位置不清楚 | 打开文件管理器，进入 Download 文件夹 |
| 导出失败 | 存储空间不足或权限不足 | 检查存储空间和权限设置 |
| 文件损坏 | 导出过程中出错 | 重新导出 |

### 导入问题

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 导入失败 | CSV 格式错误或缺少必要列 | 检查 CSV 文件格式 |
| 列匹配错误 | 列名不标准 | 修改列名为标准格式 |
| 数据不完整 | 学生姓名或学号为空 | 检查 CSV 文件数据 |
