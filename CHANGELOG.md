### 🐛 已知问题修复

#### Bug #1: Provider 数据未加载
- **问题**：重启应用后设置页面显示数据为 0
- **原因**：使用 `ref.read()` 时 Provider 可能未完成加载
- **修复**：在 build 方法中使用 `ref.watch()` 预加载数据

#### Bug #2: 统计排名显示为空
- **问题**：有学生参与点名但排名显示为空
- **原因**：评分时只更新 avgScore，未更新 callCount
- **修复**：评分时重新计算并更新 callCount

#### Bug #3: 首页点名按钮提示班级为空
- **问题**：重启后点击"开始点名"提示无班级
- **原因**：与 Bug #1 相同的 Provider 懒加载问题
- **修复**：在首页 build 中预加载班级数据
