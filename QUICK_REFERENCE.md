# 导入导出功能快速参考

## 📍 文件位置

### 导出文件
- **位置：** `/storage/emulated/0/Download/`（Android）
- **访问：** 打开文件管理器 → Download 文件夹
- **文件名：** `students_班级名_时间戳.csv`

### 导入模板
- **位置：** `/storage/emulated/0/Download/`（Android）
- **获取：** 点击"下载导入模板"按钮
- **文件名：** `import_template_时间戳.csv`

## 🚀 快速操作

### 导出学生名单
1. 班级管理 → 班级菜单 → 导出数据 → 导出学生名单
2. 文件保存到 Download 文件夹

### 导出点名记录
1. 班级管理 → 班级菜单 → 导出数据 → 导出点名记录
2. 文件保存到 Download 文件夹

### 生成统计报告
1. 班级管理 → 班级菜单 → 导出数据 → 生成统计报告
2. 文件保存到 Download 文件夹

### 导入班级和学生
1. 班级管理 → 导入按钮
2. 输入班级名称
3. 点击"下载导入模板"（可选）
4. 点击"选择 CSV 文件"
5. 从 Download 文件夹选择文件
6. 预览数据
7. 点击"确认导入"

## 📋 CSV 格式

### 导入模板格式
```csv
学生姓名,学号
张三,001
李四,002
```

### 导出学生名单格式
```csv
班级名称,学生姓名,学号,被点名次数,平均分
高一(1)班,张三,001,5,4.20
```

## ⚠️ 常见问题

| 问题 | 解决方案 |
|------|--------|
| 找不到导出文件 | 打开文件管理器，进入 Download 文件夹 |
| 导入失败 | 检查 CSV 格式，确保有标题行和有效数据 |
| 下载模板失败 | 检查存储空间和权限 |
| 导入白屏 | 检查文件格式，尝试重新下载模板 |
